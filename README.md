## ✨ Design Philosophy

This portfolio is built as a digital experience, utilizing **Glassmorphism** and high-end animations to showcase technical depth and design precision.

* **Chic Tech Hero**: A floating 3D laptop animation paired with a dynamic typing effect to immediately capture attention.
* **Coffee-to-Code**: Custom SVG coffee cup animation with CSS-animated steam particles—a tribute to the late-night sessions at **IJSE**.
* **Education Timeline**: A vertical, gradient-mapped journey tracking my path from **St. John's College** to becoming a **Full-Stack Developer**.
* **Performance First**: Built with **Vite** for lightning-fast load times and optimized asset delivery.

## 📥 Local Setup

1. **Clone the repository**
  ```bash
     git clone https://github.com/dilinichamikasilva/Dilini_Chamika_Silva.git
  ```

2.Enter the project directory
  ```bash
    cd my-portfolio
  ```

3.Install dependencies
  ```bash
  npm install
  ```

4.Launch development server
  ```bash
  npm run dev
```
